## Project Overview
Image forgery is a common cybercrime involving manipulation of digital images.  
This project aims to classify images as authentic or tampered by learning visual patterns using CNN-based feature extraction.

## Methodology
- Image preprocessing and normalization
- CNN model training for feature extraction
- Binary classification of authentic vs forged images
- Performance evaluation on test images
